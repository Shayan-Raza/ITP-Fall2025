# Assignment 3
The assignment has 4 tasks.
## Task 1
The task uses 2 functions. One function returns a new sorted string while the other sorts the string inplace.
### How to compile and execute
Linux / macOS

``
g++ task1.cpp -o app
``

``
./app
``

Windows (MinGW)

``
g++ task1.cpp -o app.exe
``

``
app.exe
``
